    The Box Model

    Creating a section to demonstrate proper use of the box model

    This is a box demonstrating the CSS box model.

    Note: The box has padding, border, and margin applied.

    Padding

    padding is the space inside an element between its content and border

    Border

    border is the line that goes around the padding and content of an element

    Margin

    margin is the space outside an element between its border and other elements

    pushes element away from each other

    This division briefly illustrate how border, margin and padding works with content.

    1. Padding: The space between the content and the border of the box.
    2. Border: The line that surrounds the padding and content.
    3. Margin: The space outside the border, separating the box from other elements.
